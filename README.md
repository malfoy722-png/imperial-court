# 上朝了

一个桌面优先的 AI 朝堂沙盒首版。每局会随机生成架空朝廷、十二名核心重臣、今日奏目与国势盘；先离线可玩，再在游戏里的“模型”页填入 MiniMax Key 点亮 AI 朝班。

## 本地运行

```bash
npm install
npm run dev:desktop
```

桌面开发窗口会同时启动本地页面。离线朝班已经支持点名、追问、准奏、驳回、转办、退朝、御书房查验、旨意簿、密报匣和自动存档。

## 校验

```bash
npm test
npm run build
npm run lint
```

## 当前边界

- 首版先做十二名核心重臣和朝堂核心状态，不做省份地图、完整战争模拟与 Steam 打包。
- 存档与本地设置走桌面 SQLite；直接打开网页开发页时会回退到浏览器本地存储。
- MiniMax 接入由桌面主进程发起。导演模型默认 `MiniMax-M2.7`，角色模型默认 `M2-her`，角色发言失败时会回落到导演层继续推进朝会。
